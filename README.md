A responsive Recipe Finder web application built using **HTML, CSS, and JavaScript** that allows users to search for delicious recipes using the MealDB API.

---
# Live Demo
https://github.com/lahari07-creator/recipe-finder-website.git

---
# Features

-  Search recipes by name
-  View complete recipe instructions
-  Display recipe image, category & region
-  Popup modal for detailed recipe view
-  Fetch real-time data using MealDB API
-  Responsive and clean UI design

---
# Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- MealDB API (https://www.themealdb.com/)

## How It Works

1. User enters recipe name in search bar.
2. JavaScript fetches data from MealDB API.
3. Recipes are displayed dynamically.
4. Clicking "View Recipe" opens a popup with full instructions.
---
##  API Used

This project uses the free MealDB API:

Search API:
https://www.themealdb.com/api/json/v1/1/search.php?s=recipe_name
